Conectar Firebase

Para que los datos sean compartidos entre el equipo, necesitas conectar un proyecto Firebase gratuito. Solo hay que hacerlo una vez — luego se guarda automáticamente.

¿Cómo obtener estos datos? Ve a console.firebase.google.com → crea proyecto → Firestore Database → Configuración del proyecto → Tu app web → verás el objeto firebaseConfig con estos campos.

Activa Firestore en modo prueba (sin auth) para que todos puedan entrar sin login. Los datos se guardan en Firestore, gratis hasta 1 GB.
