Conectar Firebase

Base de datos gratuita para sincronizar el equipo en tiempo real. Solo hay que configurarlo una vez — se guarda en el navegador.

Cómo obtenerlo: console.firebase.google.com → crea proyecto → Firestore Database (modo prueba) → Configuración del proyecto ⚙️ → Tu app web → verás el firebaseConfig.
